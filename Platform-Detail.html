Elastic Compute

Elastic Compute is a service that allows customers to initialize on demand a variety of virtual server resources including central processor (CPU), temporary memory (RAM), storage capacity (Storage) and network system (Networks) without having to invest in hardware equipment in the data center (Data Center).

CMC Elastic Compute is built on the most modern genuine hardware system from Dell; KVM-OpenStack technology solution application has been recognized by Amazon, IBM, Alibaba ... The system is located at 03 Data Centers meeting Tier 3 International standards of CMC Telecom with the only PCI DSS security certificate in Vietnam. The service exclusively uses international stable cable connection system, high bandwidth up to 100Gbps; quality commitment guarantee (SLA) 99.99%.

Customers can easily launch the service after just a few clicks; All administration and operation operations are performed via Web interface.
